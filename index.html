01Poetiq 系统解构:两层

Layer 1 · 内层 Harness(开源,已忠实复现)

generate
LLM 写程序→ verify
跑可见测试→ refine
执行反馈迭代→ vote
多专家共识

可见 I/O 对 = 免费可执行 verifier;失败时把 expected-vs-got diff 按 improving order 喂回;N 个并行专家按行为签名聚类多数投票。本质 = AlphaCodium/Reflexion + 执行接地的自修复 + verifier 上的投票。

Layer 2 · Meta-System(从未公开,我们的重建)

genome
旋钮+prompt→ proposer
LLM 读失败迹改写→ δ-margin 选择
cost-aware J→ H* transfer
冻结跨模型

外层循环只在 dev 模型上优化("learn once"),冻结 H* 原样迁移到其他模型("use everywhere")。指标:Mean Held-out Lift / Transfer Ratio / Leapfrog Rate。明确标注为我们的设计——复刻 Poetiq 秘方是 reverse-engineering 而非 replication。
